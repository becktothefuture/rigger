Safety first

Guardrails are non-negotiable.

Rigger reads first, edits later. Everything is reversible and version-control friendly. It is designed to keep your codebase clean.

  • Read-only scan before any edits
  • Confidence thresholds on suggestions
  • WorkspaceEdit for full undo
  • No remote script injection
  • Diff-first variable hooks only
  • Live preview without runtime patching
